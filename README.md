Journey Joy

Journey Joy هو تطبيق سياحي ذكي يهدف إلى تسهيل تجربة السفر من خلال توفير منصة متكاملة للمستخدمين لاستكشاف الوجهات السياحية، حجز الرحلات، وإدارة خطط السفر بكل سهولة.



المميزات

 للمستخدم:

* استكشاف المدن والوجهات السياحية
* البحث عن:

  * الفنادق 
  * المطاعم والمقاهي 
  * الأنشطة الترفيهية 
* حجز رحلات وخطط سياحية متكاملة
* الدفع الإلكتروني داخل التطبيق 
* إضافة الأماكن إلى المفضلة 
* تتبع حالة الحجوزات
* دعم تعدد اللغات (العربية / الإنجليزية)
* التكامل مع خرائط Google Maps


لأصحاب المكاتب السياحية:

* إضافة وتعديل:

  * الأماكن السياحية
  * المطاعم والمقاهي
  * الأنشطة
* إنشاء وتنظيم الرحلات
* إدارة حجوزات العملاء


لوحة تحكم المدير:

* إضافة / تعديل / حذف البيانات داخل التطبيق
* إدارة الرحلات
* مراقبة الإحصائيات:

  * عدد المستخدمين 
  * عدد الحجوزات 
  * الأرباح


 الوظائف الأساسية

* إنشاء حساب وتسجيل الدخول والخروج
* تعديل الملف الشخصي وحذف الحساب
* البحث داخل التطبيق
* حجز الرحلات
* تعديل أو إلغاء الحجز
* الدفع الإلكتروني


هدف المشروع

يهدف التطبيق إلى تحسين تجربة السفر وجعلها أكثر سهولة وتنظيمًا، من خلال جمع جميع الخدمات السياحية في منصة واحدة متكاملة.
